# **App Name**: FlowPro

## Core Features:

- User Authentication: Secure user registration and login using email/password and Google, powered by Firebase Authentication.
- Personalized Sales Profile Quiz: A mandatory interactive quiz to gather user details and preferences, storing responses in Firestore for personalized planning.
- AI-Powered Action Plan Generator: An AI tool that generates a concise and personalized sales action plan based on the user's quiz responses, focusing on practical execution.
- Premium Access Paywall: Displays a clear paywall offering monthly and lifetime subscription options to unlock the full app experience and personalized plan.
- Guided Daily Mission System: A step-by-step mission interface with daily tasks (e.g., 'Create Offer', 'Find Clients'), practical instructions, suggested content, and progress tracking saved to Firestore.
- AI Sales Mentor Chatbot: An interactive AI tool where users can ask sales-related questions and receive guidance and advice from a virtual mentor.
- Actionable Sales Resources & Progress: A centralized area providing ready-to-use sales scripts (selling, follow-up, closing) with copy functionality, alongside displaying the user's progress and completed tasks from Firestore.

## Style Guidelines:

- Primary color: A vibrant, uplifting medium purple (#8E44AD) to evoke creativity and ambition. This color provides a sense of premium quality and focus.
- Background color: A soft, muted lavender-gray (#F3EEF5), offering a clean and calming canvas that allows the primary purple and content to stand out without distraction.
- Accent color: A bright, energetic magenta (#E74C8D) to highlight calls-to-action, progress indicators, and important elements, creating a dynamic contrast and drawing user attention.
- Body and headline font: 'Inter' (sans-serif) for its modern, clear, and neutral aesthetic, which supports the app's direct, practical, and execution-focused nature for maximum readability across all content.
- Use minimalist and intuitive line icons that clearly convey actions, progress, and categories. Examples include checkmarks for completed tasks, chat bubbles for the AI mentor, and arrows for navigation.
- Employ a clean, grid-based, and mobile-first layout prioritizing essential information and calls to action. Ample whitespace will ensure a non-distracting user experience, true to its 'simple, modern, and conversion-focused' design.
- Incorporate subtle, functional animations for feedback on interactions (e.g., button presses, mission completion) and visual indication of progress, contributing to a lightweight gamified feel without overcomplicating the design.
